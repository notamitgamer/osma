# Acknowledgements

OSMA is built on the shoulders of some truly generous platforms and communities. A huge thank you to everyone who makes this project possible for free.

## is-a.dev

[is-a.dev](https://is-a.dev) provides free `*.is-a.dev` subdomains for developers — no cost, no strings attached. OSMA's home at **data.amit.is-a.dev** is made possible entirely through their service.

If you're a developer looking for a clean, free subdomain for your own project, check them out and consider [contributing to their repository](https://github.com/is-a-dev/register).

---

## Hugging Face

[Hugging Face](https://huggingface.co) powers two critical parts of OSMA:

- **Dataset Hosting** — The multi-gigabyte NPM and PyPI snapshot CSVs are stored and served from Hugging Face Datasets, making them reliably accessible without any bandwidth costs.
- **Backend Services** — Both the NPM and PyPI REST APIs are deployed as Hugging Face Spaces, providing free compute for serving millions of package records via an in-memory SQLite index.

Without Hugging Face's generous free tier, this project simply would not exist in its current form.

---

## Firebase

[Firebase](https://firebase.google.com) by Google handles hosting for the OSMA web frontend. Firebase Hosting provides fast, globally distributed static hosting with a free tier that covers OSMA's traffic needs comfortably.
